# 거기... 어때요..? - 실시간 혼잡도 채팅방

## 프로젝트 개요

"거기... 어때요..?"는 한국의 유명한 관광지와 랜드마크들의 실시간 혼잡도 정보를 공유하는 채팅방을 모아놓은 모바일 전용 웹사이트입니다. 사람들이 해당 장소를 방문하기 전에 현재 상황을 정확히 알 수 있도록 도와줍니다.

## 주요 기능

### ✅ 구현된 기능

1. **반응형 모바일 디자인**
   - iPhone에 최적화된 레이아웃
   - 핫핑크 테마의 깔끔한 디자인
   - 터치 친화적인 인터페이스

2. **실시간 랜드마크 정보**
   - 10개의 한국 유명 랜드마크 정보
   - 실시간 참여 인원수 표시
   - 위치 정보 및 설명 제공

3. **카카오톡 오픈채팅방 연동**
   - 각 장소별 채팅방 입장 버튼
   - "https://open.kakao.com/o/gDerDQug" 링크 연결

4. **자동 새로고침**
   - 30초마다 데이터 자동 업데이트
   - 실시간 참여 인원수 반영

### 📱 지원 장소

## 기술 스펙

### 사용 기술
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **CSS Framework**: 순수 CSS (모바일 최적화)
- **아이콘**: Font Awesome 6.0
- **폰트**: 시스템 기본 폰트 (iOS 최적화)

### 데이터 저장소
- **테이블 스키마**: landmarks
- **필드**: id, name, location, participants, image_url, description

### API 엔드포인트
- `GET /tables/landmarks` - 랜드마크 목록 조회
- `POST /tables/landmarks` - 새 랜드마크 추가
- `PUT /tables/landmarks/{id}` - 랜드마크 정보 수정
- `DELETE /tables/landmarks/{id}` - 랜드마크 삭제

## 사용 방법

1. **웹사이트 접속**: 모바일 브라우저에서 접속
